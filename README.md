# Digital Wellness Platform

A responsive web application built with PHP, HTML5, and CSS3 that encourages healthier digital habits through mood reflection, self-care activities, trusted wellness resources, and guided breathing exercises.

# Project Overview

Digital Wellness was created to provide users with a simple, calming, and organized space for improving their relationship with technology and supporting their mental well-being.

Instead of overwhelming users with information, the platform focuses on small daily habits that encourage mindfulness, reflection, and healthy routines.

---

# Features

- Daily Mood Journal
- Trusted Mental Wellness Resources
- Daily Self-Care Challenges
- Guided Breathing Exercise
- Daily Wellness Tips
- Responsive Layout
- Shared PHP Header, Navigation, and Footer
- Session-Based Mood Reflection
